# 🌐 GIEST Website

This repository contains the source code for the **GIEST Website**, the official online platform of GIEST Fest.  
The website is designed to provide event details, registration portals, announcements, and an engaging user experience for participants.

---

## ✨ Features
- 📅 Event Information & Schedule  
- 📝 Online Registration & Participant Management  
- 📰 Announcements & Updates Section  
- 📸 Gallery & Highlights  
- 📱 Fully Responsive UI (Mobile + Desktop)  

---

## 🛠 Tech Stack
The website is built with modern technologies to ensure performance and scalability:

- **Frontend:** React.js / Next.js, Tailwind CSS  
- **Backend:** Node.js / Express  
- **Database:** MongoDB / PostgreSQL  
- **Hosting:** Vercel / Netlify / AWS (depending on deployment)
